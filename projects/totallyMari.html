PROJECT DESCRIPTION

Interactive, bilingual, and responsive website created for the Canadian Robotics Competition (CRC) - Avia 2023. The site, inspired by 'Totally Spies!' (2001), immerses its user in a 2000's setting that narrates the Marianopolis Robotic team's journey.

Totally Mari was awarded 1st place in Website Design & 2nd place in Website Content at the Canadian Robotics Competition (CRC)

KEY FEATURES

+ Features a puzzle-solving component that serves as a gatekeeper for accessing certain pages, adding an interactive element to the user experience

+ Incorporates a name input field accessible to users, enabling personalization across the website by displaying the entered name throughout the user's browsing session

+ Extensive CSS event-driven animations, enhancing visual aesthetics and user engagement by introducing dynamic elements throughout the site
