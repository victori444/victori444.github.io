DESKTOP EMULATOR

December 2023

HTML5, CSS, JavaScript

Concordia University

Individual

View Source Code

PROJECT DESCRIPTION

Interactive website emulating MacIntosh desktop interfaces from the 1980s to the current modern interface. This project seeks to provide engaging insight on the UX/UI design transformations particular to the MacIntosh operating system, from the conception of the Macintosh 128K up to the 2020 MacBook.

KEY FEATURES

+ Variable user interface allowing users to change the desktop display according to their decade of choice

+ Animated music player, offering varying track options based on the chosen decade

+ Real-time date & time
