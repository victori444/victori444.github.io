TOTALLY MARI

2022-2023

Marianopolis Robotics

Alongside Carolyn Wu & Celia Shi

View Source Code

PROJECT DESCRIPTION

Interactive, bilingual, and responsive website created for the Canadian Robotics Competition (CRC) - Avia 2023. The site, inspired by 'Totally Spies!' (2001), immerses its user in a 2000's setting that narrates the Marianopolis Robotic team's journey.

Totally Mari was awarded 1st place in Website Design & 2nd place in Website Content

KEY FEATURES

+ Extensive user interactivity using React Context

+ Engaging CSS animations & embedded 3D models using the Google model-viewer library

+ Unique and visually appealing user experience through both design & content
