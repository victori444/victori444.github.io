View Source Code

PROJECT DESCRIPTION

Bobbing Bat is a spin on the classic game Flappy Bird. The game allows the user to travel through an infinite cave through which they must avoid stalagmites and stalactites using the spacebar, and earn points along the way!

KEY FEATURES

+ In-game points that are updated at every successfully-passed obstacle

+ Embedded music adjustable by the user, and sound effects synchronized to the player's controls

+ Custom graphics created for the scope of the game
