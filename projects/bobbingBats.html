BOBBING BAT

January 2023

C#, Unity

Personal Project

Individual

PROJECT DESCRIPTION

Bobbing Bat is a spin on the classic game Flappy Bird. The game allows the user to travel through an infinite cave, controlling a group of bats in order to avoid incoming stalagmites and stalactites using the spacebar, and earn points along the way!

KEY FEATURES

+ Progressively accumulating in-game points awarded upon overcoming each obstacle successfully

+ Integrated music player with adjustable settings for users, complemented by synchronized sound effects reacting to player actions

+ Bespoke graphics crafted exclusively for the game's design and experience
